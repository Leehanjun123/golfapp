
# 🏌️ Golf AI Coach 설치 가이드

## 📱 Android 설치 방법

### 방법 1: 웹 앱 사용 (가장 간단!)
1. Android 폰에서 Chrome 브라우저 열기
2. 주소창에 입력: https://golf-ai-coach.web.app
3. 메뉴 → "홈 화면에 추가"
4. 완료! 앱처럼 사용 가능

### 방법 2: APK 직접 설치
1. 설정 → 보안 → "알 수 없는 출처" 허용
2. APK 파일 다운로드
3. 다운로드 폴더에서 APK 파일 실행
4. 설치 완료

## 🔗 중요 정보
- 백엔드 서버: https://golfai.loca.lt
- 백엔드는 개발자 PC에서 실행 중
- 문제 발생시 개발자에게 연락
